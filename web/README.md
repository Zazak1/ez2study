# Ez2Study Web Frontend

Ez2Study AI智能学习视频生成平台的Web前端应用。

## 🚀 快速开始

### 环境要求
- Node.js 16+
- 现代浏览器 (Chrome, Firefox, Safari, Edge)

### 安装依赖
```bash
cd web
npm install
```

### 开发服务器
```bash
npm run dev
```
访问 http://localhost:3000

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 🔧 开发指南

### 添加新页面
1. 在`src/`目录下创建对应的HTML/CSS/JS文件
2. 在`main.js`中添加路由逻辑
3. 更新导航菜单

### 自定义样式
1. 在`main.css`中添加新的CSS规则
2. 使用CSS变量保持一致性
3. 遵循BEM命名规范

### 添加交互功能
1. 在`main.js`中添加事件处理函数
2. 使用模块化的代码结构
3. 添加适当的错误处理
